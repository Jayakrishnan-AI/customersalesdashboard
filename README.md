# Customer Sales Dashboard in Excel

## Overview
This project presents an interactive **Customer Sales Dashboard** built in **Microsoft Excel** using a sales dataset.  
The dashboard is designed to convert raw transactional data into clear visual insights that support business analysis and decision-making.

## Dashboard Highlights
- **Total Sales:** ₹22,97,200.86
- **Total Profit:** ₹2,86,397.02
- **Average Sales:** 229.86
- **Average Profit:** 28.66
- **Total Quantity:** 37,873

## Features
- Interactive **Slicers** for:
  - Category
  - Region
- KPI Cards for high-level performance tracking
- **Sales Trend Analysis** across cities
- **Profit Comparison** by category
- **Quantity Analysis** by shipping mode
- **Regional Sub-Category Comparison**
- **Customer Segment Distribution** using a donut chart

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Dashboard Design

## Business Insights
- **Technology** category generates the highest profit.
- **West** region shows strong sub-category performance.
- **Standard Class** contributes the highest order quantity.
- The dashboard enables quick filtering and comparison across categories and regions.
